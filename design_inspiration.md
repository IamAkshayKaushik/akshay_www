# Design Inspiration

This document outlines the design inspiration for this portfolio website.

## References

The design of this website is inspired by the following portfolios:

- **leerob.io:** The portfolio of Lee Robinson, VP of Product at Vercel. It's a great example of a content-rich, yet minimalist site.
- **joshwcomeau.com:** Josh Comeau's site is vibrant and interactive, showcasing a mastery of CSS and React.

## Color Palette

The color palette is designed to be clean, professional, and easy on the eyes. It features a dark mode theme, which is very popular among developers.

## Typography

The typography is designed to be modern and highly readable. The primary font is Inter, a modern and highly readable sans-serif font.
